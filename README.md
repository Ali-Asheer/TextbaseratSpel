# Examensuppgift: Textbaserat Minröj i C++
### Uppgiftens beskrivning
Du ska implementera ett textbaserat Minröj-spel i C++. Spelet ska vara spelbart i konsolen och innehålla grundläggande funktioner som gör det möjligt för en användare att spela spelet genom att avslöja rutor, markera minor och undvika att trampa på en mina.

## Access the game:

```bash

g++ -std=c++17 -o minesweeper main.cpp Board.cpp

./minesweeper

```
